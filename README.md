# 🚀 Job Portal (MERN Stack)

A full-stack Job Portal web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). This platform connects job seekers with employers, enabling seamless job posting, searching, and application management.

## 🌐 Live Demo
👉 https://omizzz-job-portalproject.netlify.app/

---

## 📖 Project Overview

This Job Portal allows:
- 👨‍💼 Employers to post jobs, manage listings, and view applicants
- 👨‍🎓 Job seekers to search and apply for jobs
- 🔐 Secure authentication system for both users

The platform is designed to simplify the hiring process and provide a smooth user experience.

---

## ✨ Features

### 🔑 Authentication System
- User Signup & Login
- Employer Signup & Login
- JWT-based authentication

### 🧑‍💻 Job Seeker Features
- Browse available jobs
- Search & filter jobs
- Apply for jobs
- Profile management

### 🏢 Employer Features
- Add / Edit / Delete job postings
- Manage company profile (logo, about, etc.)
- View applicants

### ⚡ Advanced Features
- Image upload (company logo)
- Protected routes (middleware)
- REST API integration
- Responsive UI
